### Key Concepts

**Customer Segment:**
- Target audience group (e.g., SMB, Enterprise)
- Has characteristics, pain points, value drivers

**Value Proposition:**
- Compelling statement of value
- Links to customer segment
- Has benefits, differentiators, proof points

**Benefit:**
- Specific advantage to customer
- Tangible (measurable) or Intangible (emotional)
- Has impact level (high/medium/low)

**AI Contextual Help:**
- AI assistant integrated at each step
- Provides suggestions and guidance
- Context-aware based on current step
